Dijkstra algoritmusa

 

Bemenet: G - gráf

Kimenet: E - hasító táblázat(CSP→CSP),T - hasító táblázat(CSP→szám)

1:

függvény Dijkstra(G : GRÁF; start : CSP)

2:

ciklus mind x ∈ G.Csúcsok

3:

L[x] ← ∞ ; E[x] ← ø

4:

S ⇐x

5:

ciklus vége

6:

L[start] ← 0

7:

ciklus amíg S ≠ ø

8:

u ← S.MinKivesz(L)

9:

ciklus mind x ∈ u.Szomszédok

10:

ha L[u] + u.Súly(x) < L[x] akkor

11:

L[x] ← L[u] + u.Súly(x)

12:

E[x] ← u

13:

elágazás vége

14:

ciklus vége

15:

ciklus vége

16:

vissza (E, L)

17:

függvény vége

 

 

Felhasznált változók és függvények:

L: Egy hasító táblázat, amelyik tárolja csúcsonként az addig talált legrövidebb út hosszát.

E: Egy hasító táblázat, amelyik csúcsonként a legrövidebb talált úthoz tartozó elõzõ csúcsot tárolja.

S: Egy prioritási sor, amely a gráf csúcsait tárolja, és a legközelebbit vissza tudja adni.

S.MinKivesz(L): Az S prioritási sorból visszaadja azt a csúcsot, amelyhez tartozó L érték minimális. Egyben törli is ezt az elemet a sorból.

u; x: CSP típusú segédváltozók.
